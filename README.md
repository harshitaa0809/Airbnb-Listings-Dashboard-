# Airbnb Dashboard Analysis

## Overview
This Tableau dashboard analyzes Airbnb listing data to identify pricing trends, bedroom distribution, revenue trends, and average prices across different zip codes.

## Tools Used
- Tableau
- Excel/CSV

## Key Insights
- Average price increases with the number of bedrooms.
- One-bedroom listings are the most common.
- Prices vary across different zip codes.
- Revenue shows an overall increasing trend throughout 2016.
